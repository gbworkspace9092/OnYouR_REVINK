# REVINK - 프리미엄 두피문신 & 헤어 스타일링 웹사이트

## 프로젝트 개요

REVINK는 프리미엄 두피문신(SMP)과 헤어 스타일링 전문샵의 웹사이트로, 고객들에게 브랜드를 알리고 예약 및 상담을 받을 수 있는 플랫폼입니다.

### 브랜드 컨셉
- **REVINK** = Revive(되살리다) + Ink(잉크)
- **철학**: "당신의 두피 위에 새겨지는 점은 외적인 변화가 아닌, 당신의 새로운 시작점입니다."
- **슬로건**: "외적 변화뿐만 아닌, 내적 변화까지"

## 주요 기능

### 1. 브랜드 소개 (About)
- REVINK 브랜드 스토리
- 철학과 슬로건
- 프리미엄 샵 소개

### 2. 서비스 안내 (Services)
- 두피문신(SMP) 서비스
- 헤어 스타일링
- 두피 케어 프로그램
- 각 서비스별 특장점 설명

### 3. 포트폴리오 (Portfolio)
- 시술 전후 비교 사진
- 고객 변화 사례
- 작업 결과물 갤러리

### 4. 고객 후기 (Reviews)
- 실제 고객들의 후기
- 평점 시스템
- 신뢰성 있는 리뷰 시스템

### 5. 예약 및 상담 (Contact)
- 온라인 예약 폼
- 카카오톡 상담 연결
- 전화 상담 연결
- 위치 정보 및 영업시간

## 사용 방법

### 로컬에서 실행
1. 파일을 다운로드합니다
2. index.html 파일을 웹 브라우저로 엽니다
3. 반응형 디자인을 확인하려면 브라우저 개발자 도구를 사용합니다

### 커스터마이징
- 색상: CSS 변수를 수정하세요
- 콘텐츠: HTML 파일의 텍스트를 수정하세요
- 이미지: 실제 이미지 URL로 교체하세요
- 연락처: 실제 연락처 정보로 업데이트하세요
